
# MagicWound
魔法创伤——闲的没事干设计的卡牌

## 项目简介
这是一个基于 C++ 的卡牌游戏项目，包含角色、卡牌、卡组等基本游戏元素。项目支持卡组创建、卡牌管理、卡组导出与导入等功能。

## 主要功能
- **角色系统**：定义角色属性（如名称、元素、技能等）及操作（如显示角色信息）。
- **卡牌系统**：支持卡牌的创建、序列化与显示，包含卡牌类型（生物卡、法术卡）、稀有度、属性（攻击、防御、生命）等机制。
- **卡组系统**：支持卡组的创建、管理、洗牌、导出与导入（通过编码实现），并可统计卡组元素分布。
- **数据库系统**：提供角色与卡牌的统一管理，支持按名称、ID、元素、稀有度等条件查询。
- **游戏管理器**：提供用户交互菜单，支持查看卡牌/角色信息、创建卡组、导出/导入卡组等功能。

## 核心特性
- 使用 Boost 实现 CRC32 校验与 Base64 编码/解码，用于卡组编码的安全性与可读性。
- 支持命令行界面交互，自动适配 UTF-8 编码（Windows 环境）。
- 卡组编码格式包含 CRC32 校验，确保导入数据的完整性。

## 编译与运行
### 编译环境
- C++17 或以上
- Boost 库（用于 CRC32 与 Base64）
- Windows（用于 UTF-8 控制台适配）

### 编译方式
项目提供 `build.bat` 脚本用于快速编译：
```bat
build.bat
```

### 运行方式
编译成功后，运行 `MagicWound.exe`：
```bat
MagicWound.exe
```

## 使用示例
1. **创建卡组**：选择菜单中的“创建卡组”选项，输入卡组名称并添加卡牌与角色。
2. **导出卡组**：在卡组创建完成后，使用“导出卡组编码”功能生成可分享的卡组编码。
3. **导入卡组**：使用“导入卡组”功能，输入编码自动还原卡组内容。
4. **查看信息**：可查看所有卡牌与角色的详细信息。

## 文件结构
- `magicwound.cpp`：核心逻辑实现。
